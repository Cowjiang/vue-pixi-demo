## 基于Vue3、PIXI、GSAP、AlloyTouch的H5一镜到底Demo

### 相关依赖
```
Vue 3.2、Vite、Pinia、Typescript、Pixi 6、GSAP 3（TimelineMax、TweenMax）、AlloyTouch
```

### 运行 dist 目录
```
npm i http-server -g

http-server
```
